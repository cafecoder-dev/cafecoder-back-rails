# CafeCoder Provisional Backend

## How to run

Ruby version: 2.6.3

Run this commands:
```shell script

# If not installed:
$ gem install bundler

$ bundle install
$ bundle exec rails server -b 0.0.0.0 -p 4000 -e development 
```

Access to http://localhost:4000
